# DOM Project 2 - Interactive Color Changer

This exercise builds a simple DOM interaction project using **HTML, CSS, and JavaScript**.

## What this project does

- Displays a title, a color box, and a **Change Color** button.
- When the button is clicked, JavaScript generates a random color.
- The background color of the box updates instantly.

## Files

- `index.html` - page structure (container, heading, box, button)
- `style.css` - basic styling and layout
- `script.js` - DOM logic and random color change behavior

## How to run

Open `index.html` in a browser, then click **Change Color** to test the interaction.
